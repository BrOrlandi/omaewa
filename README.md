> Omae wa mou shindeiru! Nani!??

Play famous "Omae wa" sound.

## Compatibility

- Linux
- Windows
- Mac

## Installing
Install globally:

    npm install -g omaewa

## Running
Run command:

    omaewa

Or import in NodeJS script:

    import omaewa from 'omaewa';

    omaewa();
